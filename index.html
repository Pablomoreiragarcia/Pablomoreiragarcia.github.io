    La Viña Wine Prediction

    Caja Mar UniversityHack Datathon 2023 / Master Final Project

    Description:
    Spain is the third largest wine producer in the world. Having an accurate forecast of the production in an agricultural campaign is essential to optimise harvesting, transfer, processing, storage and distribution.

    Based on historical production data from the vineyards of La Viña cooperative and historical weather data, this project builds machine learning models to predict the production of a campaign months before the harvest, allowing the cooperative to plan the whole process in advance.




    Injury Risk Prediction

    Degree Final Project

    Description:
    The main objective of this project is the development of Python code for the construction of a module that integrates algorithms and models available in WEKA under the Scikit-Learn interface. This module is applied to real datasets of professional indoor football players to predict muscular injuries.

    Different techniques to deal with class imbalance are evaluated, and model behaviour is explained using SHAP-based explainable AI techniques, providing interpretable insights about the factors that contribute most to injury risk.
